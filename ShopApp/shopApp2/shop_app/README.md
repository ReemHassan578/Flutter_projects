# shop_app

A Flutter project that provides a user-friendly shopping experience, featuring a bottom navigation bar for seamless navigation between key sections, including Products, Categories, Favorites, and Profile, allowing users to browse a variety of products, save favorites, and manage their profile settings efficiently.
